# 模拟面试「视频面试风」界面改造 设计文档

日期：2026-05-20
范围：纯前端，仅改动 `src/components/MockInterview.vue`（含其 `<style scoped>`）。Rust 后端零改动。

## 背景

当前模拟面试界面"做题感"太重，主要来源：
- 显式的「第 N / M 题」考试计数器；
- 每答完一题立刻弹出数字分数 + 点评；
- 每道题是独立卡片，翻到下一题后上一题消失，没有连续对话的感觉。

中期答辩需要让它看起来"真的像在面试"。

## 目标

把 interview 阶段从"答题表单"改造成"视频面试"的视觉体验，且：
- 沿用项目现有设计语言（`tokens.css` 变量、`fr-card`/`fr-btn`/`fr-chip` 等类），浅色 + Rust 橙暖色调，避免"AI 化"通用样式；
- 面试中**不显示任何分数/点评**；
- 后端数据流和命令完全不变。

## 非目标（留作后续）

- 真实摄像头 / 麦克风 / 语音转文字——本次只做视觉占位，预留扩展位置。
- 后端 prompt 调整（让面试官口吻更自然）——本次复用现有 `comment`/`follow_up` 文案。

## 设计

### 三个阶段保持不变的骨架
`setup → interview → report` 三段式不变。`setup` 配置页（考点 / 题量 / 难度）和 `report` 报告页结构基本保持，仅做与新风格一致的微调。

### interview 阶段（核心改造）

布局自上而下：

1. **顶部视频条**（双画面）
   - 左：AI 面试官画面。圆形头像（`--accent` 橙底 + "AI" 文字）居中；左上角「● 录制中」红点（`--danger`）；右上角计时器（`--font-mono` 等宽，从面试开始计时，MM:SS 累加）；左下角「AI 面试官」标签。
   - 右：候选人画面占位。`--surface-2` 底，居中显示「📷 摄像头未开启」，左下角「你」标签。为未来摄像头/语音预留。
   - 用 `fr-card` 的视觉规格（圆角 `--radius-lg`、`--shadow-sm`、`--border`）。

2. **面试官提问框**
   - `--accent-soft` 淡橙底 + 左侧 3px `--accent` 橙色边框。
   - 顶部小标签「面试官提问」（`--accent` 色），下方问题正文。
   - **打字机效果**：问题文字逐字显示，结尾光标 `▌`。出现时附带短暂"面试官思考中"指示。

3. **对话历史**
   - 已完成的问答以气泡形式留在上方，可向上滚动回看（面试官问题靠左，候选人回答靠右）。
   - 历史气泡中**不含分数**。

4. **作答区**
   - `fr-card` 容器内一个多行 `fr-input` textarea；右下角「跳过」（ghost）+「回答」（primary `--accent`）按钮。
   - 提交后：调用现有 `submit_mock_answer` → 拿到 `{score, comment, follow_up}` → **只把 `follow_up` 当面试官的自然追问显示**（不显示 score / comment）。
   - 用户补充追问回答 → 调用现有 `submit_mock_follow_up` → 进入下一题；最后一题后调用 `finish_mock_interview` 生成报告。

### report 阶段
保持现有报告结构（平均分 + 逐题点评 + AI 总结）。分数与 `comment` 点评在此处统一呈现——这是面试中被隐藏的反馈的最终落点。仅做与新风格统一的样式微调。

## 状态与数据流

复用现有 `MockInterview.vue` 的 ref 状态与 Tauri 命令，不增删后端接口。新增的纯前端状态：
- 计时器（`elapsedSeconds` + 定时器，进入 interview 启动、离开/结束清理）；
- 打字机显示用的逐字渲染状态；
- 对话历史列表（把每轮的"面试官问题 / 候选人回答 / 追问 / 追问回答"累积渲染为气泡，分数不入历史）。

`currentEvaluation` 仍保存后端返回值，但 UI 不再渲染其 `score`，`comment` 不在面试中展示（进报告）。

## 风格约束

- 不硬编码颜色，全部走 `tokens.css` 变量；
- 复用 `fr-card`/`fr-btn`/`fr-btn-primary`/`fr-btn-ghost`/`fr-chip`/`fr-input`/`fr-mono`；
- 深色模式自动跟随 `[data-theme="dark"]` 暖色变量；
- 克制动效，仅打字机 + 轻量过渡，不加渐变 / 夸张阴影 / 科技蓝。

## 验证

- `npm run tauri dev` 实机走一遍：setup → 完整答完一场 → report；
- 确认面试中全程无分数、无点评；计时器走动；打字机正常；对话历史可回看；
- 切换深色模式确认配色联动；
- 报告页分数 / 点评 / 总结正常显示。
